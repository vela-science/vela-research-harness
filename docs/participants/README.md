# Independent temporal-registration participants

Status: accepting expressions of interest. No external run has started.

Canopus seeks:

- three independent producers for one cold Vela landing task; and
- five independent reviewers for the temporal actor-registration rubric.

No prior Vela use is required. Producers should know basic Git and command-line
work. Reviewers need to read a bounded CLI report and explain its authority
meaning.

The public experiment uses:

- Vela `v0.800.22`;
- Canopus `v0.1.11`;
- immutable Git bundles and registered case facts;
- no human key inside a model, runner, browser, or harness; and
- the worksheets in this directory.

The stopped first-party Stage A v5 and v6 runs remain historical evidence. The
external runs do not continue those registrations or pool their scores.

## Producer task

Each producer receives an isolated frontier, an exact released Vela binary,
and `temporal-registration-producer-run.md`. The producer inspects roots,
runs the registered verifier, lands one Receipt, and explains why the result
remains pending. A passing run changes no accepted state or historical event.

Expected time: 45 to 90 minutes.

## Reviewer task

Each reviewer receives the registered cases and
`temporal-registration-reviewer-run.md`. The reviewer classifies thirteen
strict-verification cases and answers five authority questions before seeing
the answer key.

Expected time: 30 to 45 minutes.

## Independence

Participants disclose prior Vela or Canopus work, relationships to the protocol
team, implementation lineage, and any hidden access to fixture answers. A fresh
model session controlled by the Vela team remains first-party and receives no
independent credit.

## Support and stop rules

The facilitator may provide installation help already covered by the public
packet. Semantic hints, command repair, artifact editing, key exposure,
historical rewrite, or a false strict pass stop the run.

Do not begin from this README alone. The protocol team must assign an opaque
participant ID and freeze the exact packet, binary hash, case order, and
intervention-log root first.

To volunteer, reply to the public recruitment issue with:

- producer or reviewer;
- Git and command-line experience;
- prior Vela or Canopus involvement; and
- a repository or public profile that supports the independence declaration.
